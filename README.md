# Colliery_WebApp
Sviluppo di una web app interattiva per clienti di un'azienda di impianti fotovoltaici (colliery-sunspire-enermetal). Volto ai clienti, facendoli registrare e compilare i moduli per effettuare dei pull-out test, eseguire impianti fotovoltaici e/o agrivoltaici, con l'inserimento dei vari dati personali dei clienti
